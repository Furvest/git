# git

Игра, созданная лучшими разработчиками (по мнению разработчиков) группы ПМИ24. Проект будет из себя представлять маленькую игру.
A game, was created a best developers (developers opinion) a "ПМИ24" group. Project will be a little game.

## Краткое описание проекта

**Hell yeah game** - компьютерная игра в жанре RRPG с элементами визуальной новеллы и roguelike. Игровая карта представляет собой несколько локаций, соединенныхмежду собой данжами. Игровой процесс строится на прохождении подземелий и взаимодействии с игровыми персонажами. Игра создается 3 студентами МАУ направления ПРикладная математика и информатика.
